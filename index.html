Веб-разработка

Веб-разработка — процесс создания веб-сайта или веб-приложения. Основными этапами процесса являются веб-дизайн, вёрстка страниц, программирование на стороне клиента и сервера, а также конфигурирование веб-сервера.

Виды веб-разработки

Front-end
Back-end
Fullstack

Зоны ответсвенности

Front-end

  • Разработка пользовательской части веб-приложения
  • Настройка соединения с сервером
  • Обработка и отправка данных на сервер

Back-end

  • Разработка сервера веб-приложения
  • Обработка полученных данных с фронта
  • Сохранение данных в базе данных

Fullstack

  • Работает за двоих

Как стать разработчиком

Начать свой путь в веб разработке можно с html и css. После понимания как работает верстка можно приступить к JavaScript. Если вы хотите углубиться во front-end, то начните изучать React и Redux. Если в бэкенд, то Java или C#. Если в фуллстек разраотку, то стоит выучить React и Redux в связке с Node.js. Подробнее можете прочитать по ссылке ниже.
